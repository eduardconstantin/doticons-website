## ðŸŒ± Getting Started

Clone the repository:

```bash
git clone https://github.com/eduardconstantin/doticons-website.git
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

Open your browser and navigate to [http://localhost:3000](http://localhost:3000).
